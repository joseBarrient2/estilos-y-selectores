SELECTORES

1 - selector de etiqueta

son los que permiten aplicar estilos a todas las etiquetas de un tipo, ejem:, a todos las parrafos, div, listas desordenadas y enlaces, etc

Se aplicaron stilos a este div

Piña soy un parrafo

2 - selector de clase

los selectores de clase buscan los elemento que tengan el atributo class. Este puede encadenar a varios elemento con el mismo nombre de clase

Se aplicaron estilos la clsae a

soy clase b

3 - selector de Id

los selectores de id buscan los elemento que tengan el atributo id y que tenga el valor exacto que pide. Los elemento id son unicos, asi que no se pueden repetir.


Soy on ID unico

4 - selector Descendencia

Se puede combinar dos selectores tales que el selector combinado incluye sólo los elementos que coinciden con el segundo selector

5 - selector Hijo directo (>)

Es un selector parecido al selector descendente, pero funciona de diferente modo. Se utiliza para seleccionar un elemento que es primer hijo de otro elemento

6 - selector Hermano siguiente o adyacente (+)

Es un selector parecido al selector descendente, pero funciona de un modo diferente. Se utiliza para seleccionar un elemento hermano que esta al costado de otro elemento

7 - selector Hermanos siguientes o adyacentes (~)

Se utiliza para seleccionar un todos los hermanos directos del primer elemento que esta a su costado

8 - selector de anidamiento (,)

Es selector nos ayuda a agrupar elementos que este relacionados entre si, sin importar el nivel de profundidad del elemento seleccionado
